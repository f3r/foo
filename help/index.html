Skin/Colors How to change layout skin and colors ?



Dark Layout

Place dark.css in all of the html pages right after main.css.
dark.css could be found inside assets/css/dark.css



Dark Navigation

  • Find <header> Tag in the html files (This will be at the start of html files)
  • Add class "dark" in the <header> tag


Colors

Premade colr style sheets could be found inside assets/css/colors/
Chose desired colr and place it after main.css

Unsticky Navigation

  • Find <nav> Tag in the header of html files
  • Add class "nav-stop" in the <nav> tag

How to use flexslider

	$('#container').flexslider({
        animation: "slide",
        directionNav: false,
        controlNav: false,
        pauseOnHover: true,
        slideshowSpeed: 6000,
        direction: "horizontal" //Direction of slides
    });
More options and documentation, please check here.
